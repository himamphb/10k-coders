HTML Versus XHTML

HTML XHTML
HTML is the predominant mark up language for web pages XHTML is a family of XML languages which extend or mirror versions of HTML
It is written as tags surrounded by angle brackets XHTML requires that there be an end tag to every start tag and all nested tags must be closed in the right order.
document type is mandatory The xmlns attribute in is mandatory
All tags and attributes are not necessarily to be in lower or upper case In this, every tag and attribute should be in lower case
While using the attributes it is not necessary to mention quotes While using the attributes it is mandatory to mention quotes
Filename extension used are .html, .htm Filename extension are .xhtml, .xht, .xml
